# Security Gate — демо-GUI (Streamlit)

Красивый интерфейс для защиты кейса: живой скан + графики результативности и сравнения с аналогами.

## Запуск
```bash
source .venv/bin/activate
pip install streamlit matplotlib          # если ещё не стоят
streamlit run gui/app.py
```
Откроется в браузере (по умолчанию http://localhost:8501).

## Вкладки
- **🛡️ Скан (демо)** — выбери датасет/версию → «Запустить Security Gate». Вызывает **реальный**
  пайплайн (`python -m src.cli scan`), показывает вердикт **ALLOW/REVIEW/BLOCK**, разбор по слоям и
  подозрительные строки. Демо-сценарий: `d4 (чистый) → ALLOW`, затем `d2/d3 (отравленный) → REVIEW/BLOCK`;
  включи «Режим 2» (эталон d4) для калиброванного **BLOCK**.
- **📊 Результативность** — ROC-AUC по семействам атак, FP на чистом.
- **🆚 Аналоги** — матрица покрытия, слои, head-to-head vs cleanlab.
- **ℹ️ О решении** — проблема/угроза/цена/решение.

## Файлы
- `app.py` — UI (Streamlit).
- `gate_runner.py` — чистая логика вызова гейта и парсинга отчёта (без streamlit, тестируемо).
- графики берутся из `post_train_guard/benchmarks/figures/`.

## Совет для демо
Табличный скан идёт ~10–40 c (обучаем MLP под кейс). NLP (BERT) на CPU — минуты, для живого
показа лучше табличные. Тема — фирменный красный в `.streamlit/config.toml`.
